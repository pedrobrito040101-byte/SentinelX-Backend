# SentinelX Backend

Backend do sistema SentinelX responsável pelo monitoramento e simulação de ameaças cibernéticas em tempo real.

## Tecnologias

- Node.js
- Express
- Socket.io

## Funcionalidades

- API de alertas
- Comunicação em tempo real via WebSocket
- Simulação de eventos de segurança
- Integração com dashboard frontend

```bash
npm install
node src/server.js
